# Vue 脚手架

## 命令

### 开发服务

```bash
npm run start
```

OR

```bash
yarn run start
```

### 生产环境

```bash
npm run build
```

OR

```bash
yarn run build
```

## 版本
- Vue @ 2.5.16
- Webpack @ 4.8.1
  - Vue Style Loader @ 4.1.0
  - Html Webpack Plugin @ 3.2.0
  - Extract Text Webpack Plugin @ 4.0.0-beta.0
  - Optimize CSS Assets Webpack Plugin @ 4.0.1
  - mini CSS Extract Plugin @ 0.4.0
